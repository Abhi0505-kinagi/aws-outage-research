# Analysis of AWS Outage Impact and Cloud Reliability Measures

**Author:** Abhishek H. Kinagi  
**Institution:** PES University, Bengaluru, India  
**Year:** 2025  

This repository contains the research paper and supporting materials analyzing the AWS outage incident and its implications for distributed cloud systems.
